# Developer Intern - Job Description

## About Weaverse

Weaverse is a cutting-edge platform that empowers Shopify merchants to build stunning, high-performance online stores. We're passionate about e-commerce innovation and creating tools that make a real difference for businesses worldwide.

| | |

---

## What You'll Learn

Throughout this internship, you'll gain hands-on experience with modern web development technologies and practices:

### Frontend Development
- **React** - Building dynamic, component-based user interfaces
- **TypeScript** - Writing type-safe, maintainable code
- **TailwindCSS & ShadcnUI** - Creating beautiful, responsive designs
- **Preact Signals** - Modern state management techniques

### Backend Development
- **Node.js & Express.js** - Building RESTful APIs and server-side applications
- **PostgreSQL & Prisma ORM** - Database design and management
- **Zod** - Schema validation and type safety

### E-commerce Specialization
- **Shopify Platform** - Understanding the Shopify ecosystem
- **Hydrogen** - Shopify's React-based framework for custom storefronts
- **Weaverse** - Our visual page builder and theme customization platform

### Development Tools & Practices
- **Git & GitHub** - Version control and collaboration
- **Vite** - Modern build tooling
- **Code Review** - Best practices and team collaboration
